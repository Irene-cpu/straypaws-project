# Stray Paws 🐾

A multi-page responsive website designed for a stray cat adoption initiative.

## 📖 About The Project
<img width="1470" height="800" alt="Screenshot 2026-02-24 at 8 56 57 PM" src="https://github.com/user-attachments/assets/4fa8e3c1-44c2-40e9-8054-686689cdc578" />

Stray Paws is a static website built to raise awareness, promote adoption, and encourage donations for stray cats. 

This project was created to practice:

- Semantic HTML structure
- CSS layout (Flexbox)
- Responsive design using media queries
- Proper file structure management
- Git workflow (init, commit, push)
- Deployment using GitHub Pages

---

## ✨ Features

- Homepage
- Adoption page
- Donation page
- Awareness page
- Contact page
- Responsive layout for mobile and desktop
- Clean folder structure

## 🛠 Built With

## 📱 Responsive Design

The layout adjusts based on screen size using CSS media queries to ensure usability on mobile devices.
